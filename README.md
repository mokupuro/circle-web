## 開発の流れ

- 「タスク」カラムから自分が取り組むタスクを選んで、Assigneesで自分のアカウンをを選択する
- 「作業中」カラムにタスクを移動する
- ブランチを切って、開発をする
- pull requestのdescriptionにissueの番号（#1とか#2のように）を書いてプルリクする
- 「作業中」カラムのタスクで対応が終わったものから「確認待ち」カラムに移動する
- レビューしてもらってマージしたら、Close issueボタンを押して「作業完了」カラムに移動されてたらOK！
